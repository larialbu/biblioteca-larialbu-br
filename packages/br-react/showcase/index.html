br-validators

Validadores

Funções puras para validação de CPF, CNPJ, CEP, telefone e mais.

npm install @larialbu/br-validators
Demo — isCPF()

Digite um CPF para validar com o algoritmo oficial

Código
import { isCPF } from '@larialbu/br-validators';

isCPF('123.456.789-09'); // true
isCPF('111.111.111-11'); // false
isCPF('000.000.000-00'); // false
br-masks

Máscaras

Funções de formatação para inputs com máscaras brasileiras.

npm install @larialbu/br-masks
Demo — cpfMask()

Digite números e veja a máscara sendo aplicada

Resultado: —
Código
import { cpfMask } from '@larialbu/br-masks';

cpfMask('12345678909');
// → '123.456.789-09'

// Use em qualquer input:
<input onChange={e =>
  setValue(cpfMask(e.target.value))
} />

npm install @larialbu/br-utils

// → 'R$ 1.000.000,00'
